null undefined 和 boolean

发表于 2019-06-20

null和undefined

  • null 表示空值，即该处的值现在为空
  • undefined表示 “未定义”

boolean

boolean 代表 “真“ 和 ”假“ 两个状态。”真” 用true表示，“假” 用false表示。boolean只有这两个值。

返回布尔值得运算符

  • 前置逻辑运算符： ‘ (Not)
  • 相等运算符：===，!==，==，!=
  • 比较运算符：>，>=，<，<=

六个“false”值

  • undefined
  • null
  • false
  • 0
  • NaN
  • "" 或 '' （空字符串）
